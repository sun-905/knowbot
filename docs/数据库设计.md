## 表结构

### users — 用户表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | |
| phone | VARCHAR(20) | UNIQUE, NULLABLE | 手机号 |
| email | VARCHAR(255) | UNIQUE, NULLABLE | 邮箱 |
| password_hash | VARCHAR(255) | NOT NULL | bcrypt 哈希 |
| nickname | VARCHAR(100) | DEFAULT '' | 昵称 |
| avatar_url | VARCHAR(500) | DEFAULT '' | 头像 |
| daily_quota | INT | DEFAULT 100 | 每日提问上限 |
| is_admin | BOOLEAN | DEFAULT FALSE | 是否管理员 |
| is_active | BOOLEAN | DEFAULT TRUE | 是否启用 |
| created_at | DATETIME | DEFAULT NOW() | |
| updated_at | DATETIME | DEFAULT NOW() ON UPDATE | |

### sessions — 会话表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | |
| user_id | BIGINT | FK→users.id, NOT NULL | |
| kb_id | BIGINT | NULLABLE | 关联知识库，空=全部 |
| title | VARCHAR(200) | DEFAULT '新对话' | |
| status | ENUM('active','closed') | DEFAULT 'active' | |
| created_at | DATETIME | DEFAULT NOW() | |
| updated_at | DATETIME | DEFAULT NOW() ON UPDATE | |

### messages — 消息表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | |
| session_id | BIGINT | FK→sessions.id, ON DELETE CASCADE | |
| role | ENUM('user','assistant','system') | NOT NULL | |
| content | TEXT | NOT NULL | |
| intent | VARCHAR(50) | NULLABLE | 意图分类结果 |
| references_json | JSON | NULLABLE | 引用的知识来源 |
| token_count | INT | DEFAULT 0 | |
| created_at | DATETIME | DEFAULT NOW() | |

### knowledge_bases — 知识库表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | |
| name | VARCHAR(200) | NOT NULL | |
| description | TEXT | DEFAULT '' | |
| is_default | BOOLEAN | DEFAULT FALSE | |
| created_at | DATETIME | DEFAULT NOW() | |
| updated_at | DATETIME | DEFAULT NOW() ON UPDATE | |

### knowledge_docs — 知识库文档表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | |
| kb_id | BIGINT | FK→knowledge_bases.id, DEFAULT 1 | |
| filename | VARCHAR(500) | NOT NULL | |
| file_type | ENUM('txt','md','pdf') | NOT NULL | |
| file_size | INT | DEFAULT 0 | |
| chunk_count | INT | DEFAULT 0 | 切片数量 |
| status | ENUM('processing','ready','failed') | DEFAULT 'processing' | |
| error_msg | VARCHAR(500) | NULLABLE | 失败原因 |
| created_at | DATETIME | DEFAULT NOW() | |
| updated_at | DATETIME | DEFAULT NOW() ON UPDATE | |

### feedbacks — 反馈表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | |
| message_id | BIGINT | FK→messages.id | |
| user_id | BIGINT | FK→users.id | |
| rating | ENUM('like','dislike') | NOT NULL | |
| comment | VARCHAR(500) | NULLABLE | 文字反馈 |
| created_at | DATETIME | DEFAULT NOW() | |

### daily_usage — 每日用量表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK | |
| user_id | BIGINT | FK→users.id | |
| usage_date | DATE | NOT NULL | |
| count | INT | DEFAULT 0 | 当日提问次数 |
| created_at | DATETIME | DEFAULT NOW() | |

---

## 初始化

建表语句位于 `backend/sql/init.sql`。Docker Compose 首次启动 MySQL 时自动执行。包含 7 张表和 1 条默认知识库记录：

```sql
INSERT INTO knowledge_bases (name, description, is_default)
VALUES ('默认知识库', '系统默认知识库', TRUE);
```
