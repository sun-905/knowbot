# AI 智能客服系统 — 项目说明

> 基于 RAG（检索增强生成）的企业级 AI 智能客服系统

## 项目定位

将企业知识库文档转化为可对话的智能助手。用户用自然语言提问，系统从知识库中检索相关内容，由 LLM 生成有据可查的回答。

## 技术架构

| 层 | 技术 | 说明 |
|---|------|------|
| 前端 | React 19 + Ant Design 6 + Vite 8 | SPA, 暗色科幻主题 |
| 后端 | FastAPI (Python 3.10+) | 异步 Web 框架 |
| 数据库 | MySQL 8.0 | 业务数据持久化 |
| 向量库 | Qdrant | 文档向量存储与检索 |
| 大模型 | DeepSeek (主) + Qwen (备) | LLM 生成，双模型兜底 |
| 嵌入 | BGE-large-zh-v1.5 (BAAI) | 文本向量化 |
| 重排序 | BGE-Reranker-v2-m3 | Cross-Encoder 精排 |

## 核心功能

- **RAG 智能问答**：混合检索（向量+BM25）+ 重排序 + 上下文守护 + LLM 流式生成
- **意图识别**：三级级联（L1规则→L2向量→L3大模型），置信度 <0.7 追问澄清
- **知识库管理**：上传文档自动解析、切片、向量化，支持多知识库
- **会话管理**：独立 Session，多轮对话上下文，历史记录查询
- **反馈系统**：点赞/踩 + 可选文字反馈
- **管理后台**：问答统计、反馈统计、意图分布、折线图
- **反幻觉机制**：分层摘要、类型排序、生成后三层验证、SYSTEM_PROMPT 硬约束

## 快速开始

详见 `运行指南.md`。

```bash
# 1. 启动基础设施
docker compose up -d mysql qdrant

# 2. 配置 API Key
cp .env.example .env
# 编辑 .env，填入 DEEPSEEK_API_KEY

# 3. 启动后端
cd backend && pip install -r requirements.txt
uvicorn app.main:app --port 8000

# 4. 启动前端
cd frontend && npm install && npm run dev
```
